# NasaAmber-IOS

**This app uses NASA API. Photos from this API are listed. Photos can be listed according to vehicle cameras. Photo details are displayed when the photo is pressed.**
## Screens

1. Intro
2. Home
   - Curiosity
   - Opportunity
   - Spirit
3. Filter
4. Detail
